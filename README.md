# Diabetes Prediction Web App

A Machine Learning web application built using Streamlit and Support Vector Machine (SVM) to predict whether a person is diabetic or not.

## Features

- Streamlit Web Interface
- Real-Time Prediction
- Machine Learning Model
- SVM Classifier

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
